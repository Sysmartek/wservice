Modulo DataBase vai ser simples pois iremos trabalhar com Spring Data
explicação sobre DAO e Repositorio
	Diferenças entre DAO e Repositórios
	
	DAO colocamos nossos códigos SQL, HQL e serve para gerenciar nossas informações no Banco de Dados, tudo a nível de query
	aqui devo criar toda a consulta detalhada a nível de sql , com Likes, Unios, etc..
	Tenho que criar toda a implementação da nossa classe concreta (implementação de um interface)
	
	vs
	
	Repositorio iremos trabalhar com consultas baseadas em nome de métodos
	Faço a mesma consulta baseado no nome sem fazer a query  (exemplo:)
	cria consultas a nível de execução
	Também dá para criar a consulta a nível de SQL caso precisar
	Nao preciso criar a implementação da classe concreta, o Spring-Data-Jpa já FAZ  todo o CRUD da minha interface
